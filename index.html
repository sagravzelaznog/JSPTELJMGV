Sesión 1 — Variables, Tipos de Datos y console.log

En esta sesión aprenderás a almacenar información en variables, conocer los tipos de datos fundamentales y mostrar resultados en la consola.

1.1 — Tu primer programa: Hola Mundo

// Línea 1: Mostramos un mensaje en la consola del navegador
console.log("¡Hola Mundo!");
Línea 1 console.log() es una función que imprime mensajes en la consola del navegador. El texto entre comillas es una cadena de texto (string).

1.2 — Declarar variables con let

// Línea 1: Declaramos una variable llamada "nombre" y le asignamos un texto
let nombre = "Carlos";

// Línea 2: Declaramos una variable numérica para la edad
let edad = 17;

// Línea 3: Mostramos las variables usando template literals (backticks)
console.log(`Mi nombre es ${nombre} y tengo ${edad} años`);
Línea 1 let declara una variable que puede cambiar de valor. "Carlos" es un string (cadena de texto).
Línea 2 17 es un number (número). JavaScript detecta el tipo automáticamente.
Línea 3 Los template literals usan backticks (`) y ${variable} para insertar variables dentro del texto.

1.3 — Constantes con const

// Línea 1: Declaramos una constante (no puede cambiar de valor)
const PI = 3.14159;

// Línea 2: Declaramos el radio de un círculo
let radio = 5;

// Línea 3: Calculamos el área del círculo
let area = PI * radio * radio;

// Línea 4: Mostramos el resultado
console.log(`El área del círculo es: ${area}`);
Línea 1 const declara una constante. Si intentas reasignarla (PI = 3), JavaScript mostrará un error.
Línea 2 Usamos let porque el radio podría cambiar en el futuro.
Línea 3 El operador * multiplica valores. La fórmula del área es π × r².
Línea 4 Mostramos el resultado interpolando la variable area en el texto.

1.4 — Los tipos de datos principales

// Línea 1: String (cadena de texto)
let texto = "Hola";

// Línea 2: Number (número entero o decimal)
let numero = 42;

// Línea 3: Boolean (verdadero o falso)
let esEstudiante = true;

// Línea 4: Undefined (variable sin valor asignado)
let sinValor;

// Línea 5: Null (valor vacío intencional)
let vacio = null;

// Línea 6: typeof nos dice el tipo de dato de una variable
console.log(typeof texto);         // "string"
console.log(typeof numero);        // "number"
console.log(typeof esEstudiante);  // "boolean"
console.log(typeof sinValor);      // "undefined"
Línea 1 String: texto encerrado en comillas dobles, simples o backticks.
Línea 2 Number: cualquier número, ya sea entero (42) o decimal (3.14).
Línea 3 Boolean: solo puede ser true o false. Muy usado en condiciones.
Línea 4 Undefined: cuando declaras una variable pero no le das valor.
Línea 5 Null: representa "nada" o "vacío" de forma intencional.
Línea 6 typeof es un operador que devuelve el tipo de dato como string.

1.5 — Operadores aritméticos

// Línea 1: Suma
let suma = 10 + 5;          // Resultado: 15

// Línea 2: Resta
let resta = 10 - 3;         // Resultado: 7

// Línea 3: Multiplicación
let multi = 4 * 3;          // Resultado: 12

// Línea 4: División
let division = 20 / 4;      // Resultado: 5

// Línea 5: Módulo (residuo de la división)
let residuo = 10 % 3;       // Resultado: 1

// Línea 6: Exponente (potencia)
let potencia = 2 ** 3;      // Resultado: 8 (2 elevado a la 3)

// Línea 7: Mostramos todos los resultados
console.log(suma, resta, multi, division, residuo, potencia);
Línea 1-4 Operadores básicos: + suma, - resta, * multiplicación, / división.
Línea 5 % devuelve el residuo. 10 ÷ 3 = 3 con residuo 1. Útil para saber si un número es par (n % 2 === 0).
Línea 6 ** es el operador de potencia. 2 ** 3 es lo mismo que 2×2×2 = 8.
Línea 7 console.log() puede recibir múltiples argumentos separados por comas.

Sesión 2 — Condicionales: if, else y switch

Las estructuras condicionales permiten que tu programa tome decisiones. El código se ejecuta solo si se cumple una condición.

2.1 — Estructura if / else

// Línea 1: Definimos la edad del usuario
let edad = 16;

// Línea 2: Evaluamos si la edad es mayor o igual a 18
if (edad >= 18) {
    // Línea 3: Se ejecuta SOLO si la condición es verdadera
    console.log("Eres mayor de edad");
} else {
    // Línea 4: Se ejecuta SOLO si la condición es falsa
    console.log("Eres menor de edad");
}
Línea 1 Creamos la variable edad con valor 16.
Línea 2 if evalúa la expresión entre paréntesis. >= significa "mayor o igual que". Si es true, ejecuta el bloque siguiente.
Línea 3 Este código solo se ejecuta si edad >= 18 es verdadero.
Línea 4 else captura el caso contrario. Si la condición es falsa, se ejecuta este bloque.

2.2 — Múltiples condiciones con else if

// Línea 1: Definimos la calificación
let calificacion = 85;

// Línea 2: Evaluamos en qué rango cae la calificación
if (calificacion >= 90) {
    // Línea 3: Excelente
    console.log("Calificación: A — Excelente");
} else if (calificacion >= 80) {
    // Línea 4: Bueno
    console.log("Calificación: B — Bueno");
} else if (calificacion >= 70) {
    // Línea 5: Regular
    console.log("Calificación: C — Regular");
} else {
    // Línea 6: Reprobado
    console.log("Calificación: F — Reprobado");
}
Línea 1 Almacenamos la calificación como número.
Línea 2 Primero verifica si es ≥ 90. Si no, pasa al siguiente else if.
Línea 3-5 Cada else if se evalúa en orden. Solo se ejecuta el primer bloque cuya condición sea verdadera.
Línea 6 El else final captura cualquier caso no cubierto por las condiciones anteriores.

2.3 — Operadores de comparación

// Línea 1: Igualdad estricta (compara valor Y tipo)
console.log(5 === 5);      // true
console.log(5 === "5");    // false (número vs string)

// Línea 2: Igualdad débil (compara solo valor, NO recomendado)
console.log(5 == "5");     // true (convierte tipos automáticamente)

// Línea 3: Diferente estricto
console.log(5 !== "5");    // true

// Línea 4: Mayor, menor, mayor/igual, menor/igual
console.log(10 > 5);      // true
console.log(3 < 1);       // false
console.log(5 >= 5);      // true
console.log(4 <= 3);      // false
Línea 1 === es la igualdad estricta: compara valor y tipo. Siempre usa esta.
Línea 2 == convierte tipos antes de comparar. Esto causa errores difíciles de encontrar. Evítalo.
Línea 3 !== verifica que sean diferentes en valor o tipo.
Línea 4 Operadores de orden: >, <, >=, <=.

2.4 — Operadores lógicos: AND, OR, NOT

// Línea 1: Definimos dos condiciones
let edad2 = 20;
let tieneCredencial = true;

// Línea 2: AND (&&) — ambas condiciones deben ser verdaderas
if (edad2 >= 18 && tieneCredencial) {
    console.log("Puede votar");
}

// Línea 3: OR (||) — al menos una condición debe ser verdadera
let esFinDeSemana = false;
let esVacaciones = true;
if (esFinDeSemana || esVacaciones) {
    console.log("¡No hay clases!");
}

// Línea 4: NOT (!) — invierte el valor booleano
let lloviendo = false;
if (!lloviendo) {
    console.log("Podemos salir al patio");
}
Línea 1 Preparamos las variables que usaremos en las condiciones.
Línea 2 && (AND): el bloque se ejecuta solo si ambas condiciones son true.
Línea 3 || (OR): el bloque se ejecuta si al menos una condición es true.
Línea 4 ! (NOT): invierte el valor. Si lloviendo es false, !lloviendo es true.

2.5 — Estructura switch

// Línea 1: Definimos el día de la semana (1-7)
let dia = 3;

// Línea 2: switch compara el valor con cada case
switch (dia) {
    case 1:
        // Línea 3: Si dia es 1
        console.log("Lunes");
        break;
    case 2:
        console.log("Martes");
        break;
    case 3:
        // Línea 4: dia es 3, así que entra aquí
        console.log("Miércoles");
        break;
    case 4:
        console.log("Jueves");
        break;
    case 5:
        console.log("Viernes");
        break;
    default:
        // Línea 5: Si ningún case coincide
        console.log("Fin de semana");
}
Línea 1 La variable dia almacena un número que representa el día.
Línea 2 switch compara dia con cada case usando igualdad estricta (===).
Línea 3 Si dia === 1, ejecuta console.log("Lunes").
Línea 4 break detiene la ejecución. Sin él, seguiría ejecutando los siguientes case.
Línea 5 default funciona como el else del switch: se ejecuta si ningún case coincide.

Sesión 3 — Ciclos y Arrays

Los ciclos permiten repetir acciones. Los arrays almacenan listas de elementos. Juntos son una herramienta muy poderosa.

3.1 — Ciclo for

// Línea 1: El ciclo for tiene 3 partes: inicio, condición, incremento
for (let i = 1; i <= 5; i++) {
    // Línea 2: Este bloque se repite 5 veces
    console.log(`Vuelta número: ${i}`);
}
// Resultado: Vuelta número: 1, 2, 3, 4, 5
Línea 1 let i = 1 → inicia el contador en 1. i <= 5 → se repite mientras i sea ≤ 5. i++ → suma 1 a i después de cada vuelta.
Línea 2 El bloque entre {} se ejecuta en cada iteración. i cambia de valor en cada vuelta.

3.2 — Ciclo while

// Línea 1: Iniciamos un contador
let contador = 0;

// Línea 2: while se repite MIENTRAS la condición sea verdadera
while (contador < 3) {
    // Línea 3: Mostramos el valor actual
    console.log(`Contador: ${contador}`);

    // Línea 4: IMPORTANTE — incrementamos para evitar ciclo infinito
    contador++;
}
// Resultado: Contador: 0, 1, 2
Línea 1 Declaramos contador fuera del ciclo porque while no tiene sección de inicio.
Línea 2 while verifica la condición antes de cada vuelta. Si es false desde el inicio, nunca se ejecuta.
Línea 4 contador++ es esencial. Sin esto, contador siempre sería 0 y el ciclo nunca terminaría (ciclo infinito).

3.3 — Arrays (arreglos)

// Línea 1: Un array almacena múltiples valores en una sola variable
let frutas = ["Manzana", "Plátano", "Naranja", "Uva"];

// Línea 2: Accedemos a elementos por su índice (empieza en 0)
console.log(frutas[0]);   // "Manzana"
console.log(frutas[2]);   // "Naranja"

// Línea 3: .length nos dice cuántos elementos tiene
console.log(frutas.length); // 4

// Línea 4: Agregamos un elemento al final
frutas.push("Fresa");

// Línea 5: Eliminamos el último elemento
frutas.pop();

// Línea 6: Mostramos todo el array
console.log(frutas);
Línea 1 Los arrays se crean con corchetes []. Los elementos se separan con comas.
Línea 2 Los índices empiezan en 0. El primer elemento es [0], el segundo [1], etc.
Línea 3 .length es una propiedad que devuelve la cantidad de elementos.
Línea 4 .push() añade un elemento al final del array.
Línea 5 .pop() elimina y devuelve el último elemento.

3.4 — Recorrer un array con for

// Línea 1: Creamos un array de estudiantes
let estudiantes = ["Ana", "Luis", "María", "Pedro"];

// Línea 2: Recorremos el array con un ciclo for
for (let i = 0; i < estudiantes.length; i++) {
    // Línea 3: Mostramos cada estudiante con su número
    console.log(`${i + 1}. ${estudiantes[i]}`);
}
// Resultado: 1. Ana, 2. Luis, 3. María, 4. Pedro
Línea 1 Array con 4 nombres.
Línea 2 Iniciamos en 0 (primer índice) y terminamos en length - 1 (último índice). Usamos < y no <=.
Línea 3 i + 1 muestra el número humano (empezando en 1). estudiantes[i] accede al elemento en la posición i.
